# My-Excel-Projects-Repository
# 📊 Excel Retail & Sales Analysis Project

## 📌 Overview
This project showcases practical Excel skills developed during a Data Technician bootcamp, using real-world **retail and sales datasets**. The focus was on data cleaning, analysis, and visualisation to extract meaningful business insights.

---

## 🎯 Objectives
- Analyse retail sales data to identify trends and patterns  
- Apply key Excel formulas and functions  
- Build dynamic summaries using PivotTables  
- Create visual dashboards for data storytelling  

---

## 🛠️ Skills & Tools Demonstrated

### 📈 Data Analysis & Formulas
- `SUM` – Calculate total sales and commissions  
- `SUMIF` – Conditional aggregation based on criteria  
- `AVERAGE` – Determine mean values  
- `AVERAGEIF` – Analyse averages under specific conditions  
- `DATE`, `MONTH`, `YEAR` – Extract and manipulate date-based data  
- `UNIQUE` – Identify distinct values in datasets  
- `VLOOKUP` – Retrieve and match data across tables  

---

### 🔍 Data Preparation
- Filtering datasets to isolate key insights  
- Sorting data (e.g. largest to smallest, custom ordering)  
- Cleaning and structuring raw data for analysis  

---

### 📊 PivotTables
- Built PivotTables to summarise sales by:
  - Region / County  
  - Product category  
- Aggregated metrics such as total sales volume  
- Enabled quick comparisons across multiple dimensions  

---

### 🧠 Advanced Logic
- Applied the `SWITCH` function to categorise sales performance:
  - **High** (>600)
  - **Medium** (300–600)
  - **Low** (<300)

---

### 📉 Data Visualisation
- Created charts to present insights clearly:
  - Bar charts  
  - Column charts  
  - Summary dashboards  
- Highlighted trends in product performance and regional sales  

---

## 📊 Key Insights
- Identified top-performing regions and products  
- Analysed customer purchasing patterns  
- Categorised sales performance for decision-making  
- Demonstrated how data can inform business strategy
