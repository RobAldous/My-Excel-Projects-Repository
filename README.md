# Excel Retail & Sales Analysis Project

## Overview
This project showcases practical Excel skills developed during a Data Technician bootcamp, using real-world **retail and sales datasets**. The focus was on data cleaning, analysis, and visualisation to extract meaningful business insights.

## Objectives
- Analyse retail sales data to identify trends and patterns  
- Apply key Excel formulas and functions  
- Build dynamic summaries using PivotTables  
- Create visual dashboards for data storytelling  

## Skills & Tools Demonstrated

### Data Analysis & Formulas
- `SUM` – Calculate total sales and commissions  
- `SUMIF` – Conditional aggregation based on criteria  
- `AVERAGE` – Determine mean values  
- `AVERAGEIF` – Analyse averages under specific conditions  
- `DATE`, `MONTH`, `YEAR` – Extract and manipulate date-based data  
- `UNIQUE` – Identify distinct values in datasets  
- `VLOOKUP` – Retrieve and match data across tables
